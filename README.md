# Personal Website — Travis Kim

A static personal site introducing my work in product management and AI platforms. Built to feel calm, considered, and a little bit warm — see [`style-guide.md`](./style-guide.md) for the design philosophy.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Home / landing page |
| `my-site.html` | About + portfolio |
| `videos.html` | Selected talks and demos |

## Local preview

No build step. Open any page directly in a browser:

```bash
open index.html
```

Or serve the folder with a one-liner if you want a real local URL:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Editing

Changes go through a pull request to `main`. See the style guide before touching layout, color, or copy tone.
